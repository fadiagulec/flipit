FlipIt

Privacy Policy

Last updated: May 14, 2026

Short version

FlipIt does not collect personal information, does not use cookies for tracking, does not run analytics scripts, and does not sell or share data with third parties. The Chrome extension only opens FlipIt with the URL of the page you're currently on, and only when you click the FlipIt button.

What the Chrome extension does

The FlipIt Chrome extension injects a single floating button (“Rate with FlipIt”) onto post pages on Instagram, TikTok, YouTube, LinkedIn, Threads, Twitter/X, and Facebook. When you click that button (or the FlipIt toolbar icon), it opens a new tab to flipit-app.netlify.app with the current page's URL appended as a query parameter so FlipIt knows which post to analyze.

That is the entire behavior. The extension:

  • Does not read the contents of pages you visit.
  • Does not scrape post text, comments, DMs, or messages.
  • Does not run in the background — it has no background worker.
  • Does not send data to any server from the extension itself.
  • Does not store anything in cloud storage. The only local state is a single “hide for this session” flag in sessionStorage, which is cleared when the tab closes.
  • Does not use cookies, advertising IDs, or fingerprinting.

What the FlipIt web app does

When you submit a URL (whether typed in directly or passed from the extension), FlipIt fetches a public preview of the post via third-party extraction services (Apify, oEmbed) and sends a description of the post to an AI model (Anthropic Claude and/or OpenAI) to produce a viral-potential rating and a rewritten script. The post URL is processed only for as long as it takes to return a result and is not retained in a user database.

The web app at flipit-app.netlify.app stores your flip count, free-trial state, and (if you've purchased Pro) your access token in your browser's localStorage. This data stays on your device — it is never transmitted to our servers or any third party.

Information we do not collect

  • Names, emails, addresses, phone numbers, or any other personal identifiers.
  • Account login credentials for Instagram, TikTok, or any other platform.
  • Health, financial, location, or authentication information.
  • Personal communications.
  • Browsing history.

Third-party services

The FlipIt web app uses these third parties only to fulfill your request:

  • Netlify — hosts the web app and the serverless functions that proxy your URL request. Netlify may log standard request metadata (IP, user agent) as part of normal web hosting.
  • Apify — extracts the public post preview from the URL you submit.
  • Anthropic and/or OpenAI — runs the AI model that scores and rewrites the post. Only the post content and your prompt are sent; no user identity is attached.

Permissions the extension requests

  • activeTab — lets us read the URL of the tab you're currently on only when you click the FlipIt toolbar icon. We use this URL to open FlipIt with the right post pre-filled.
  • Host permissions for instagram.com, tiktok.com, youtube.com, linkedin.com, threads.net, facebook.com, twitter.com, and x.com — required for the content script to inject the floating button on those sites. The script only injects a button; it does not read or modify the page's data.

Children

FlipIt is not directed to children under 13 and does not knowingly collect data from anyone.

Changes to this policy

If we ever change what the extension or web app does with data, we will update this page and bump the “Last updated” date.
